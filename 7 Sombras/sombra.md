# Sombra

para darle un efecto de sombra a una caja usaremos las siguientes  propiedades:

```css
.mi-caja{
	box-shadow: 2px 4px 15px 0px #000;
	/* 	    x   y  desen borde color*/
}

.mi-text{
	text-shadow: 2px 2px 14px #fff;
	/* 	      x  y  desen color*/
}

.mi-box__ratate{
	transform: rotate(45deg);
}

```
